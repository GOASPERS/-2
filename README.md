# Инструмент визуализации графа зависимостей  
**Этап 1 — Минимальный прототип с конфигурацией**

## Общее описание

Данное приложение представляет собой консольную программу, которая в будущем будет использоваться для визуализации графа зависимостей программных пакетов.  
На первом этапе реализован минимальный прототип, который позволяет пользователю задавать параметры работы через командную строку и выводит их на экран в удобном формате.

Этот этап нужен для того, чтобы создать основу проекта, проверить корректность работы аргументов командной строки и обработки ошибок.  
В дальнейшем на следующих этапах будет добавлена логика анализа зависимостей, построения графа и создание изображения в формате SVG.

## Настраиваемые параметры

Программа принимает несколько параметров, которые пользователь указывает при запуске:

- **--package** — имя анализируемого пакета.  
- **--url** — адрес репозитория (нужен, если используется реальный режим).  
- **--path** — путь к локальному тестовому репозиторию (нужен для тестового режима).  
- **--mode** — режим работы программы: `real` (реальный) или `test` (тестовый).  
- **--version** — версия анализируемого пакета.  
- **--output** — имя выходного файла, в котором позже будет сохранено изображение графа зависимостей (например, `graph.svg`).

Если пользователь запускает программу в режиме `real`, нужно обязательно указать параметр `--url`.  
Если режим `test`, то вместо этого нужно указать `--path`.
